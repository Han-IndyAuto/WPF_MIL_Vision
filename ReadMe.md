# 📄 WPF MIL 라이브러리를 이용한 비전프로그램 개발

## 📅 2025년 11월 20일
**작성자:** indy  
**검토자:** indy

### 1. 개요
- WPF 기반 MIL 라이브러리를 이용하여 영상처리 프로그램 개발 진행

### 2. 진행 내용
- ImgView 영역과 속성 영역을 세로줄 GridSplit을 추가하여 좌/우 확대 축소를 가능하게 수정.
- 알고리즘 선택과 선택된 알고리즘 속성 사이에 가로 줄 GridSplit을 추가하여 상/하 늘리고, 줄일 수 있도록 수정.
- 가로/세로 GridSplit 이 떨어져 있는 부분을 붙어 보이게 수정. GropBox 들의 왼쪽 Margin 을 5 pixel 추가.
